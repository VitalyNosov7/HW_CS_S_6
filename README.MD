## Домашнее задание к Уроку 6. Исключения

1. Доработайте класс калькулятора способным работать как с целочисленными так и с дробными числами.

2. *Дап задание необязательное: задействовать перегрузку операторов (предлагаю +=, *=, -=, /= вместо тех функций которые уже реализованы)
(реализация с семинара в архиве в материалах, вторая реализация в program.cs файле)
---

## Решение:

Для решения поставленной задачи, в методах этой программы,  был использован тип double. 

Программа работает следующим образом:

1.	В методе Main(), класса Program, создается экземпляр класса  (LaunchCalculator launcherCalculator = new LaunchCalculator()). Далее вызывается метод 
 launcherCalculator.LauncherControllerCalculator(), который выводит приветственное сообщение и запускает контроллер калькулятора (класс  ControllerCalculator()), вызывая его кнструктор.

2.	Запущеный контроллер (класс  ControllerCalculator()) создаёт экземпляр класса обработчика команд (ComandHandler comandHandler = new ComandHandler()). В конструкторе (класс  ControllerCalculator()) запускается и начинает свою работу метод обработчика команд (comandHandler.LaunchHandler()).
	
3.	В методе (LaunchHandler()) обработчика команд (класс ControllerCalculator())  запускается цикл while(true), в котором производятся следующие действия:
						
	3.1.  Вывод "Меню комманд";		
		
	3.2.  Получение и обработка комманд вводимых пользователем;
	
	3.3.  Получение и обработка числовых значений вводимых пользователем;
	
	3.4.  Вызывается  метод из класса (class OperationCalculator) в зависимости от того, какая операция была выбрана пользователем;
	
	3.5.  Выводится результат рассчёта;
	
	3.6.  Выводится предложение продолжить работу в программе или выйти. В зависимости от выбранных пользователем действий производится соответствующие действия. Если выбрано продолжение, тогда работа в цикле(while(true)) продолжается. Если выбрано завершение работы  тогда работа в цикле(while(true)) завершается.
